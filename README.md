# PKL-Performance-Analysis-Dashboard-Tableau
PKL Tableau Dashboard is an interactive data visualization project built using Tableau to analyze Pro Kabaddi League real time (PKL) data. The dashboard provides insights into tournament performance through Team Analysis, Player Analysis, Match Analysis, and Event Analysis. It includes KPIs, team rankings, win percentage analysis, top raiders and defenders, raid vs tackle contribution, venue analysis, and match trends. The project utilizes Tableau relationships, calculated fields, filters, parameters, and advanced visualizations to transform raw data into meaningful business insights.
